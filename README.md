# React + Vite + Tailwind Project

Це мій третій проект для портфоліо: React + Vite + Tailwind CSS 3.4.17 + lodash.

## Як запустити

1. Встановити залежності:
   npm install

2. Запустити сервер:
   npm run dev

3. Відкрити браузер на:
   http://localhost:5173/

## Структура проекту

- `src/` - React компоненти та CSS
- `public/` - статичні файли
- `index.html` - HTML-шаблон
- `package.json` - залежності та скрипти
- `tailwind.config.js` / `postcss.config.js` - конфігурація Tailwind
